## 🚀 接下来的步骤

### 1️⃣ 修改数据库配置

编辑 `app/config.py` 文件，修改以下配置：

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',           # ← 改成你的MySQL用户名
    'password': 'your_password',  # ← 改成你的MySQL密码
    'database': 'your_database',  # ← 改成你的数据库名
    'charset': 'utf8mb4'
}
```

### 2️⃣ 安装Python依赖包

在PowerShell中执行：

```powershell
cd "d:\数据库\小组大作业\app"
pip install -r requirements.txt
```

或者直接安装：

```powershell
pip install streamlit pymysql pandas openpyxl
```

### 3️⃣ 运行应用

**方法1：使用启动脚本（推荐）**
```powershell
cd "d:\数据库\小组大作业\app"
.\start.ps1
```

**方法2：直接运行**
```powershell
cd "d:\数据库\小组大作业\app"
streamlit run app.py
```

应用将在浏览器中自动打开：http://localhost:8501

## 🐛 常见问题

**Q: 启动失败？**
A: 检查是否安装了所有依赖包，运行：`pip install -r requirements.txt`

**Q: 无法连接数据库？**
A: 检查config.py中的数据库配置，确认MySQL服务已启动

**Q: 页面显示为空？**
A: 可能是数据库中没有数据，检查History和Budget表是否有数据

**Q: 导入错误？**
A: 确保在app目录下运行：`cd app` 然后 `streamlit run app.py`
